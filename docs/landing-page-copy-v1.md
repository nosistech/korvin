# KORVIN Landing Page Copy v1

## Hero

KORVIN

Self-hosted AI agents for low-cost hardware.

Voice-first. Memory-persistent. Model-agnostic. No lock-in.

## Subheading

KORVIN is an open-source AI agent framework for builders who want control over models, memory, infrastructure, and data.

## Primary Calls to Action

- View on GitHub
- Install CLI
- Read Quickstart

## Install Command

npm install -g @nosistech/korvin@latest
korvin --help

## Why KORVIN

- Self-hosted control over your agent stack
- Model-agnostic routing instead of one-provider lock-in
- Persistent memory as a core architecture layer
- Voice-first direction with safe local setup today
- Built for low-cost hardware and practical deployment paths

## What Works Today

- Public npm package: @nosistech/korvin
- korvin init local setup generation
- korvin init --voice placeholder voice preparation
- korvin init --yes safe-default setup
- korvin doctor local setup checks
- Command reference documentation
- Configuration documentation
- Quickstart documentation
- Protected remote access guidance

## Current Safety Boundaries

The npm CLI is local-only. It does not install services, configure public ports, request provider keys, read secrets, write secrets, set up a VPS, configure Cloudflare, configure Telegram, install LiteLLM runtime services, configure dashboard services, or install systemd services.
