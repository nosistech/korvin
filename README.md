# Korvin

**Self-hosted AI agent framework. Voice-first. Security-native. No lock-in.**

Korvin installs on your own machine or a $5 VPS in under 10 minutes.
You provide your own API keys. You control your own data.
NosisTech never touches your messages, credentials, or billing.

---

## What Korvin Does
- Talks to you by voice — like an always-on smart assistant
- Answers questions, runs research, drafts documents, manages your inbox
- Works through Telegram, WhatsApp, Discord, and Signal
- Self-patches against security threats using sandboxed OWASP monitoring
- Auto-generates skills from experience
- Keeps its own documentation up to date

## Quick Start
> **Note:** Korvin is in active development. npm package coming soon.
    npm install @nosistech/korvin
    korvin init
The setup wizard handles everything.
